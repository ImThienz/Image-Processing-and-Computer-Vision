+ Tạo GUI (nếu có).

2. Nộp bài: gửi qua email baotn@vaa.edu.vn các file sau (ghi tiêu đề nội dung khi gửi mail).

+ Slides bài thuyết trình.
+ Code demo (nộp folder project python).
* Nén 2 tài liệu trên thành 1 file nén rồi gửi qua email.

3. Thời gian nộp: trước 12 tiếng lúc thuyết trình nhóm: Ví dụ: 12h trưa ngày 07/10/2024 thuyết trình, thì phải nộp đủ các tài liệu trước 0h ngày 07/10/2024.

Nộp trễ:
+ Từ 0h đến trước 6h sáng: trừ 30% số điểm.
+ Từ 6h sáng đến trước 8 sáng: trừ 50% số điểm.
+ Sau 8h sáng: 0 điểm.

4. Thuyết trình nhóm:

+ Thành viên vào trễ trừ 1 điểm.
+ Thành viên không có mặt sẽ 0 điểm.
